# Food Delivery Platform

## Concept
Application de livraison avec 3 utilisateurs :
- Client
- Restaurant / Pizzeria
- Livreur

## Objectif
Vendre un abonnement mensuel aux restaurants et aux livreurs.

## Fonctionnement
1. Le client entre sur le site web
2. Il choisit un restaurant
3. Il consulte le menu
4. Il passe commande
5. Le restaurant reçoit la commande
6. L’application notifie les livreurs proches
7. Un livreur accepte la commande
8. Le livreur récupère la commande
9. Livraison au client
10. Commande terminée

## Monétisation
- Restaurants : abonnement mensuel
- Livreurs : abonnement ou commission
- Clients : gratuit
